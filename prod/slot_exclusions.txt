# EMTAC production slot deployment exclusions

# ------------------------------------------------------------

# Purpose:

# These items should NOT be copied from dev into prod slots.

#

# Production slots should contain app runtime code only.

# Shared/mutable stores live outside the slot.

#

# KEEP in slot:

# - .venv

# - .env

# - ai_emtac.py

# - wsgi.py

# - **init**.py

# - blueprints

# - modules

# - static

# - templates

# - utilities

# - Database junction to E:\emtac\Database

#

# NOTE:

# .venv and .env are listed below so deploy-copy does not overwrite them.

# They should already exist in the slot and be preserved.

#

# NOTE:

# "Database" is excluded from copy because the production slot should use

# a junction:

#

# E:\emtac\prod\slots<slot>\Database -> E:\emtac\Database

#

# Do not copy a physical Database folder into a production slot.
